À propos de GestionStock

GestionStock est une plateforme complète et moderne de gestion de stock conçue pour les entreprises de toutes tailles. Notre mission est de simplifier la gestion des produits, catégories et niveaux de stock en fournissant une interface intuitive et des outils puissants.

Nos fonctionnalités

Gestion complète
Gérez vos produits, catégories et niveaux de stock en temps réel avec alertes automatiques.
Sécurité renforcée
Authentification sécurisée avec BCrypt et gestion des rôles pour contrôler l'accès.
Tableaux de bord
Vue synthétique avec statistiques, alertes et indicateurs clés en un coup d'œil.

Notre stack technique

GestionStock est construit avec les technologies les plus robustes et modernes :

  • Jakarta Server Faces (JSF) — Framework web moderne pour interfaces dynamiques
  • Hibernate ORM — Gestion optimisée de la base de données
  • MySQL 8.0 — Base de données relationnelle haute performance
  • WildFly — Serveur d'applications Java robuste
  • PrimeFaces — Composants UI avancés et responsifs
  • CDI — Injection de dépendances pour une architecture propre

Notre vision

Nous croyons que la gestion du stock ne devrait pas être compliquée. GestionStock simplifie les processus complexes et fournit aux entreprises les outils dont elles ont besoin pour prospérer. Notre objectif est de devenir la solution de gestion de stock privilégiée pour les entreprises de toutes tailles.
